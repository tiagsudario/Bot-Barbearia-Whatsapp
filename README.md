# Bot de Barbearia Whatsapp 
-SISTEMA DE ATENDIMENTO AUTOMATICO PARA BARBEARIA VIA WHATSAPP WEB

## Funcionalidades 
-Responde clientes automaticamente
-Mostra Serviços disponiveis 
-Informa Valores
-Realiza agendamentos
-Atendimento 24hrs

## Tecnologias utilizadas
-Node.js
-whatsapp-web.js
-qrcode-terminal

## !!!COMO RODAR O PROJETO!!!
1- INSTALE AS DEPENDECIAS
'''bash
npm install
2-Execute o bot
-No terminal digite: node bot.js
3- Escaneia o QRCODE com o whatsapp pelo celular
-Aguardde aparecer no terminal "BOT ONLINE"

OBS: Para parar de rodar o bot e so desligar o pc, desconectar o whatsapp web ou dar CTRL+C no terminal, Caso de erro para conectar no wpp e somente reiniciar a internet ou fechar e abrir o whatsapp no celular
